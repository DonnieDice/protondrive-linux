# Browser Console Logs

## Purpose

Stores browser console logs from the Electron renderer process for debugging.

## Files

Console logs are saved here during development for debugging purposes.

## Usage

Logs are automatically captured when running in development mode.

## .gitkeep

The `.gitkeep` file ensures this directory is tracked by git even when empty.
